사용법

그때 그 한 판을, 다시. 그냥 보는 것도, 내 경기를 올리는 것도, 가입 하나 없이 됩니다.

두 가지 방법

Just Watch

그냥 보고 싶다면

  1. 설치도 가입도 필요 없어요. 아카이브를 열어 경기를 둘러보세요.
  2. 카드를 누르면 영상이 열립니다. 위쪽 정렬·필터로 원하는 경기를 골라요.
  3. 영상 아래 하이라이트 점을 누르면 그 장면으로 바로 점프합니다.
  4. 같은 경기를 여러 명이 올렸다면 관전 시점을 바꾸거나 분할 보기로 여러 시점을 동시에 봐요.
  5. 마음에 들면 좋아요와 코멘트를 남겨요. 로그인 없이 됩니다.
Upload

내 경기를 올리고 싶다면

  1. 다운로드 페이지에서 ENCORE 레코더를 받습니다.
  2. 실행하면 작은 창이 하나 떠요. 그대로 켜두기만 하면 됩니다.
  3. 스타크래프트를 평소처럼 플레이하세요.
  4. 게임이 끝나면 자동으로 녹화 → 분석 → 업로드까지 알아서 합니다. 올린 사람은 게임 속 내 이름으로 자동 표시돼요.
  5. 잠시 뒤 아카이브에 경기가 올라와요. 끝!

로그인 없이 — 이름이 곧 계정

ENCORE엔 가입도 비밀번호도 없어요. 레코더가 경기를 녹화할 때 게임 속 당신 이름을 읽어, 그 이름이 그대로 당신 계정이 돼요. 같은 이름이면 어느 PC에서 올려도 같은 계정으로 모입니다.

  • 웹에서 로그인 — 레코더의 [갤러리] 버튼을 누르면 이 사이트가 열리면서 자동으로 로그인됩니다. 따로 칠 코드도 없어요.
  • 나의 게임 — 로그인하면 위 메뉴에 나의 게임이 생겨요. 내가 올린 경기가 모여 있고, 거기서 표시 이름 변경·로그아웃도 할 수 있어요.
  • 삭제 — 지우고 싶은 경기는 나의 게임에서 삭제하면 됩니다. 본인 경기만 지울 수 있어 안전해요(다른 사람 경기는 못 지웁니다).
  • PC 두 대 이상 — 같은 이름이면 자동으로 같은 계정이라 그냥 올리면 됩니다. 두 번째 PC에서 삭제까지 하려면 레코더끼리 한 번 연결(설정 → 계정)하세요. 대부분은 필요 없어요.

이름은 녹화 파일에서 읽는 값이라, 삭제 같은 민감한 동작만 계정 비밀키로 안전하게 보호됩니다. 보기·좋아요·댓글은 로그인 없이 그대로 됩니다.

레코더를 실행하면

받은 파일을 실행하면 아래처럼 작은 창이 하나 뜹니다. 복잡한 설정은 없어요 — 그냥 켜두면 됩니다. 한 판 하고 나면 오른쪽 위에 내 이름이 표시돼요(그게 계정이에요).

  • 갤러리 — 이 아카이브를 열고 자동 로그인합니다.
  • 폴더 열기 — 내 PC에 저장된 원본 녹화 폴더를 엽니다.
  • 업로드 — 폴더엔 있는데 아직 안 올라간 경기를 한 번에 올립니다.
  • 재분석 — 이미 올린 경기를 최신 분석으로 다시 계산해 갱신합니다.

스타를 켜면 왼쪽 은색 불빛이 빨간색으로 바뀌며 상태가 녹화 중이 되고, 끝나면 분석 중·업로드를 거쳐 다시 대기 중으로 돌아갑니다. 그동안 창은 최소화해 둬도 괜찮아요.

녹화 설정 — 대부분 그대로 두면 돼요

레코더의 설정은 모두 자동이 기본이고, 자동이 보통 가장 좋습니다. 캡처 방식·모니터는 이제 자동으로 처리돼서 따로 만질 게 없어요. 특별한 경우에만 아래를 바꾸세요.

화질
자동이면 원본 해상도 그대로 녹화해요. 파일 용량을 줄이고 싶으면 720·480으로 낮출 수 있어요.
인코더
자동이면 NVIDIA(NVENC)가 있을 때 그래픽카드로, 없으면 CPU로 인코딩해요. 녹화 중 게임이 끊기면 여기를 확인하세요.

자주 묻는 질문

로그인은 어떻게 하나요?
레코더에서 [갤러리]를 누르면 이 사이트가 열리며 자동으로 로그인됩니다. 가입·비밀번호·코드 입력이 없어요. 녹화한 게임 속 이름이 그대로 계정입니다(한 판도 안 했다면 먼저 한 판 녹화하세요).
내가 올린 경기를 지우려면?
로그인하면 위 메뉴에 나의 게임이 생겨요. 거기서 지울 경기의 삭제를 누르면 됩니다. 본인 경기만 지울 수 있어요.
표시 이름을 바꾸고 싶어요
나의 게임 페이지에서 표시 이름 변경을 누르면 돼요. 계정은 그대로(게임 이름 기준) 유지되고 보이는 이름만 바뀝니다.
PC를 바꿨거나 두 대를 써요
스타 이름이 같으면 자동으로 같은 계정이라 그냥 올리면 됩니다. 두 번째 PC에서 삭제까지 하려면 레코더 설정 → 계정에서 한 번 연결하세요. 대부분은 필요 없어요.
처음 켰더니 Windows가 위험하다고 막아요
개인이 만든, 서명 없는 프로그램이라 그래요. 바이러스가 아니라 처음 보는 프로그램이라 띄우는 경고입니다. 추가 정보 → 실행을 누르면 됩니다. 백신이 더 세게 막으면 예외(허용)로 등록해 주세요.
게임이 끝났는데 갤러리에 바로 안 보여요
녹화본을 분석하고 클라우드에 올리는 데 잠깐 걸려요(보통 1~2분). 조금 기다렸다 새로고침하면 나옵니다.
예전에 했던 경기도 올릴 수 있나요
레코더 폴더에 녹화본이 남아 있으면 업로드 버튼으로 한 번에 올릴 수 있어요. 분석을 최신 기준으로 다시 하려면 재분석을 누르세요.
맥이나 휴대폰에서도 되나요
올라온 경기를 보는 건 맥·휴대폰 어디서든 돼요. 다만 녹화·업로드와 내 경기 관리(삭제 등)는 레코더가 있는 Windows PC에서 [갤러리]로 들어가야 합니다.
영상은 어디에 저장되나요
공개 클라우드 저장소에 올라갑니다. 이 사이트 주소를 아는 사람이면 누구나 볼 수 있어요. 보여주고 싶지 않은 경기는 올리지 않으면 됩니다.
녹화 화면이 까맣게 나와요
레코더가 캡처 방식을 자동으로 바꿔 가며 맞춰요. 그래도 까맣다면 게임을 테두리 없는 창(전체창) 모드로 두면 대부분 해결됩니다.
NVIDIA 그래픽카드인데 녹화가 안 돼요
드라이버가 오래되면 그럴 수 있어요. 다운로드 페이지 안내대로 최신 드라이버를 설치해 주세요. 구버전에선 녹화가 시작되지 않습니다.
영상은 공개 클라우드에 올라가고, 이 사이트 주소만 있으면 누구나 볼 수 있어요. 녹화·관리는 Windows에서, 보기는 어디서나. 막히거나 궁금한 게 있으면 만든이에게 알려주세요.
