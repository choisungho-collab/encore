사용 매뉴얼

레코더를 켜두고 리그 오브 레전드를 플레이하면 끝입니다. 화면이 무엇을 뜻하는지, 설정은 어떻게 바꾸는지, 막힐 때는 어떻게 하는지 안내합니다.

레코더 화면 녹화 흐름 녹화 설정 아카이브 보기 로그인 자주 묻는 질문 다운로드

레코더 화면 한눈에

Ready / Recording
지금 상태. 왼쪽 점과 글씨로 보입니다. Ready는 게임 대기 중, Recording은 녹화 중이며 옆에 경과 시간(mm:ss)이 같이 흐릅니다. 녹화 중에는 창 맨 위에 얇은 빨간 줄이 켜져, 다른 창에 가려도 알 수 있습니다.
1080p · NVENC
지금 화질·인코더. 오른쪽에 현재 설정이 칩으로 표시됩니다. 눌러서 바로 설정을 열 수 있습니다.
☁ Cloud
아카이브 연결 상태. Cloud면 업로드 준비 완료, Key needed면 service_key가 빠진 상태입니다. 업로드가 끝나면 잠깐 Uploaded ✓가 뜹니다.
Archive · Open folder
Archive는 웹 아카이브 열기, Open folder는 녹화 폴더 열기. 오른쪽 두 아이콘은 ⚙ 설정과 ▤ 로그입니다.
창 닫기 ✕
✕를 누르면 종료됩니다. 게임 중엔 켜둔 채로 두시고, 잠깐 치워두고 싶으면 최소화(─)를 쓰세요 — 작업표시줄에 남아 계속 녹화합니다.

녹화 흐름 4단계

  1. 레코더를 켜둡니다. 한 번 실행해 두면 떠 있다가 게임을 감지합니다. 창은 닫지 말고 켜둔 채로 두세요.
  2. 리그 오브 레전드를 플레이합니다. 챔피언 선택이 시작되면 자동으로 녹화가 켜지고, 화면이 Recording으로 바뀝니다. 밴픽부터 게임 종료까지 한 영상에 담깁니다.
  3. 게임이 끝나면 멈춥니다. 녹화가 자동으로 종료되고, 잠시 뒤 Riot 공식 기록으로 전적이 분석됩니다.
  4. 아카이브에 자동 업로드. 영상과 전적이 웹 아카이브에 올라갑니다. Archive 버튼으로 바로 확인하세요.

녹화 설정 ⚙ Settings

기본값(Auto)이 대부분 최선입니다. 창의 ⚙를 누르면 열리고, 바꾸면 다음 녹화부터 적용됩니다.

Quality
Auto · Source · 1080p · 720p · 480p. Auto는 최상 화질. 저장 공간이나 업로드가 부담되면 720p로 낮추세요.
Encoder
Auto · GPU(NVENC) · CPU(x264). Auto는 NVIDIA가 있으면 GPU로 녹화해 게임이 안 끊깁니다. GPU가 없으면 자동으로 CPU로 넘어갑니다. 캡처 방식과 모니터는 레코더가 알아서 잡으니 고를 필요가 없습니다.

아카이브에서 보기

내 경기 찾기

아카이브 상단 검색에서 소환사명이나 챔피언으로 찾을 수 있습니다.

멀티 시점

같은 게임을 여럿이 녹화하면 한 카드로 묶입니다. 경기 안에서 시점 탭으로 자리를 바꿔가며 같은 교전을 다시 봅니다.

전적 분석

영상 옆에 승패·KDA·CS·골드 추이·오브젝트가 Riot 공식 데이터로 정리됩니다.

로그인 자동 · 비밀번호 없음

회원가입도 비밀번호도 없습니다. 레코더의 Archive(보관함) 버튼을 누르면 브라우저가 이미 로그인된 상태로 열립니다. 로그인하면 내가 올린 영상의 제목 수정·삭제를 할 수 있어요.

어떻게 로그인되나

레코더가 게임 중에 내 Riot ID를 읽어 둡니다. Archive를 누르면 그 Riot ID로 일회용 로그인 링크를 만들어 브라우저로 넘겨, 비밀번호 없이 바로 로그인됩니다.

한글 닉네임도 OK

닉네임이 한글이어도 자동 로그인이 정상 동작합니다. 갤러리에는 한글 닉네임이 그대로 표시돼요.

여러 PC도 한 계정

계정은 PC가 아니라 내 Riot ID 기준이에요. 어느 PC에서든 그 Riot ID로 한 판 하면 그 PC에서도 자동 로그인되고, 여러 PC에서 올린 영상이 모두 한 계정으로 묶입니다. Riot API 연동도 필요 없어요.

계정은 Riot ID 기준이라, 같은 Riot ID로 플레이한 PC는 어디서든 같은 보관함으로 로그인됩니다 — 집·다른 PC에서 올린 영상이 한 계정으로 모여요.

로그인은 레코더의 Archive 버튼으로만 됩니다(웹에서 아이디로 직접 로그인하는 화면은 당분간 없어요). 로그인되면 — 레코더는 내 Riot ID를 알고 있고, 웹은 오른쪽 위에 내 닉네임과 로그아웃이 떠서 로그인 상태를 바로 확인할 수 있어요. 로그아웃 상태면 그 자리에 로그인 안 됨 상태가 보이고, 클릭하면 로그인 방법 안내가 뜽니다.

자주 묻는 질문

녹화가 안 켜져요

대부분 그래픽 드라이버가 오래된 경우입니다. NVIDIA 드라이버를 최신으로 올리면 거의 해결됩니다. 캡처 방식은 레코더가 자동으로 고르니 따로 만질 게 없습니다.

영상이 아카이브에 안 올라가요

화면 우상단이 ☁ Cloud인지 확인하세요. 인터넷이 잠깐 끊기면 영상을 로컬에 보관했다가, 연결되면 자동으로 다시 올립니다.

경기 수가 이상해요

아카이브는 15초마다 자동으로 갱신됩니다. 방금 끝낸 게임이 안 보이면 잠시 후 새로고침해 주세요.

화면이 검게 녹화돼요

레코더는 전체화면도 잡히는 WGC 방식을 기본으로 씁니다. 그래도 검게 나오면 대개 그래픽 드라이버 문제이니 최신으로 올려주세요.

처음 켰더니 한참 멈춰 있어요

최초 1회는 영상 도구(ffmpeg)를 자동으로 받아옵니다. 1~3분 기다리면 Ready로 바뀝니다.

영상 용량이 부담돼요

아카이브에는 화질은 1080p 그대로 두되 용량을 줄인 영상이 올라가고, 원본 고화질은 PC의 녹화 폴더(Open folder)에 그대로 남습니다. 더 줄이고 싶으면 ⚙ 설정 → Quality를 720p로 낮추면 됩니다.

로그인이 안 되거나 내 영상 관리가 안 떠요

갤러리는 꼭 레코더의 Archive 버튼으로 여세요 — 주소창에 직접 입력하면 로그인 정보가 안 넘어갈 수 있습니다. 내 Riot ID로 한 판 플레이해 영상이 ☁ Cloud로 한 번 올라가면, 그 뒤 Archive를 누를 때마다 자동 로그인됩니다(Riot API 연동 없이도 됩니다).
