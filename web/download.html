내 스타크래프트 경기를 자동으로 녹화하고, 리플레이를 분석해서 아카이브에 올려주는 Windows 프로그램입니다. 설치 과정 없이 압축만 풀면 바로 실행돼요.

권장 사양

운영체제
Windows 10 / 11 (64-bit)
그래픽카드
NVIDIA GeForce 권장 — 하드웨어 인코딩(NVENC)으로 게임 끊김 없이 녹화돼요. 내장 그래픽이나 AMD에서도 동작하지만 CPU 부담이 커집니다.
그래픽 드라이버
최신 버전 필수 — 오래된 드라이버에선 녹화가 시작되지 않습니다. 바로 아래에서 업데이트하세요.

그래픽 드라이버 준비

NVIDIA 드라이버를 최신으로

가장 자주 막히는 부분이에요. 드라이버가 오래됐으면 녹화가 시작되지 않습니다. NVIDIA 앱을 설치하면 그래픽카드를 자동으로 인식해 최신 드라이버를 한 번에 깔아줍니다 — 가장 간편한 방법이에요.

NVIDIA 앱으로 자동 설치 드라이버 직접 검색

시작하기 3단계

  1. 위 레코더 받기 버튼으로 sc_recorder.zip을 내려받습니다.
  2. 내려받은 sc_recorder.zip의 압축을 풀고, 생긴 sc_recorder.exe를 더블클릭하세요. 첫 실행 땐 ffmpeg 같은 도구를 자동으로 받아오니 잠깐 기다리면 됩니다.
  3. 작은 창이 뜨면 끝! 그대로 켜두고 스타크래프트를 평소처럼 즐기세요. 게임이 끝날 때마다 자동으로 녹화·분석·업로드돼 아카이브에 올라옵니다.

실행하면 어떤 화면이 나오는지, 버튼이 뭔지 궁금하다면 매뉴얼에 그림과 함께 정리해 뒀어요.

처음 실행할 때

압축 풀기 전 “차단 해제” — 내려받은 sc_recorder.zip을 우클릭 → 속성 → 아래쪽 차단 해제에 체크한 뒤 압축을 풀면 경고가 줄어듭니다.

“확인되지 않은 앱” 경고 — 서명되지 않은 개인 제작 프로그램이라 Windows가 한 번 경고를 띄울 수 있어요. 추가 정보 → 실행으로 진행하면 됩니다.

백신이 막을 때 — 드물게 백신이 강하게 막으면 예외(허용) 목록에 sc_recorder.exe를 등록해 주세요.

스마트 앱 컨트롤(SAC)이 아예 차단할 때 — 새로 올라온 빌드가 평판을 쌓기 전이라 그럴 수 있어요. 최신 빌드가 안 보이거나 막히면 저장소 Actions 탭에서도 받을 수 있습니다.
