myPENTA 레코더

리그 오브 레전드를 켜두면 알아서 녹화하고, 게임이 끝나면 Riot 공식 기록으로 전적을 분석해 아카이브에 올립니다. 설치 과정 없이, 압축을 풀면 나오는 exe 하나만 실행하면 됩니다.

레코더 받기 (.zip) Windows 10 / 11 · 무료 · 압축 풀고 실행

무엇을 하나요

자동 녹화

챔피언 선택이 시작되면 녹화를 켜고, 게임이 끝나면 멈춥니다. 밴픽부터 담기며, 따로 누를 게 없습니다.

Riot 기록 분석

승패·KDA·CS·골드·아이템·룬, 골드 추이와 오브젝트까지 공식 데이터로 정리됩니다.

멀티 시점

같은 게임을 여럿이 녹화하면 자동으로 묶여, 한 교전을 여러 자리에서 다시 봅니다.

시작하기 3단계

  1. 받은 penta_recorder.zip의 압축을 풉니다. 폴더 없이 penta_recorder.exe 파일 하나만 나옵니다.
  2. penta_recorder.exe를 더블클릭합니다. 처음 한 번은 영상 도구(ffmpeg)를 자동으로 받아오니 1~2분 기다리면 됩니다. 따로 설정할 건 없습니다.
  3. 리그 오브 레전드를 플레이하세요. 게임이 끝나면 영상과 전적이 아카이브에 자동으로 올라갑니다. 창은 켜둔 채로 두면 됩니다.

설정 선택사항

따로 만들 파일도, 입력할 정보도 없습니다. 압축을 풀고 실행하면 클라우드 연결·전적 분석·업로드가 모두 자동으로 동작합니다. (Riot ID도 게임에서 자동으로 인식합니다.)

화질이나 인코더를 직접 바꾸고 싶을 때만, 레코더 창의 ⚙ 설정에서 고르면 다음 녹화부터 적용됩니다.

처음 실행할 때

압축 풀기 전 "차단 해제" — 내려받은 penta_recorder.zip을 우클릭 → 속성 → 아래쪽 차단 해제에 체크하고 적용한 뒤 압축을 풀면 경고가 줄어듭니다.

"확인되지 않은 앱" 경고(SmartScreen) — 서명되지 않은 개인 제작 프로그램이라 Windows가 한 번 경고를 띄울 수 있습니다. 추가 정보 → 실행으로 진행하면 됩니다.

"스마트 앱 컨트롤"이 아예 차단할 때 — 실행 옵션조차 없이 막히면 스마트 앱 컨트롤(SAC)이 켜진 PC입니다. 설정 → 개인정보 및 보안 → Windows 보안 → 앱 및 브라우저 컨트롤 → 스마트 앱 컨트롤 설정에서 끌 수 있습니다. (한 번 끄면 다시 켜려면 Windows 재설치가 필요하니 신중히 결정하세요.) 대부분의 PC는 SAC가 꺼져 있어 이 화면을 보지 않습니다.

백신이 막을 때 — 드물게 강하게 막으면 예외(허용) 목록에 penta_recorder.exe를 등록해 주세요.
