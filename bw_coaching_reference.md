# 브루드워 고급 코칭 레퍼런스 — 진부함 탈출판

> 목적: "일꾼 더 뽑아 / 확장 더 해 / 컨트롤 연습해" 같은 **누구나 아는 말** 대신,
> 숫자·타이밍·메커닉·이유가 붙은 **실전 코칭**을 하기 위한 참고서.
> 출처: Liquipedia StarCraft Brood War, TeamLiquid, StrategyWiki 등 영문 고급 데이터 합성.
> 대상: 스타크래프트 리마스터(브루드워). 앙코르 리플레이 코치 리포트의 진단 기준으로도 활용 가능.

---

## 0. 코칭 철학 — "진부한 조언"을 "진단 가능한 조언"으로

모든 지적은 **① 구체적 숫자/타이밍/메커닉 + ② 이유(why)** 를 붙인다.

| 진부한 조언 | 업그레이드된 조언 |
|---|---|
| "매크로 신경 써" | "6:00에 미네랄 400 떠 있었어요 — 게이트 하나 더 짓거나 프로브를 끊지 말았어야" |
| "컨트롤 연습해" | "뮤탈이 한 볼리에 마린 한 기에 다 꽂혀서 41뎀 낭비됐어요. 홀드로 2~3기에 분산하세요" |
| "정찰 더 해" | "상대 2팩 벌처인 걸 못 보고 확장만 해서 런바이에 프로브 12기 잃었어요" |
| "확장 더 해" | "앞마당 먹을 타이밍에 병력이 3게이트 분량인데도 안 나가서 상대가 3가스 먼저 폈어요" |

**리플레이 진단 순서(5축)** — 항상 이 순서로 본다:
1. **빌드 실행** — 서플 블록, 일꾼 끊김, 건물 타이밍이 표준에서 얼마나 밀렸나
2. **정찰 → 반응** — 상대 빌드를 언제 파악했고, 그에 맞춰 빌드를 바꿨나 (파악만 하고 안 바꾸면 무의미)
3. **교전** — 마이크로 + 병력 조합 + 포지셔닝(플랭크/사거리). 셋 중 뭐가 무너졌나
4. **자원 운영** — 미네랄/가스 유휴 자원, 확장 타이밍, 병력 대비 일꾼 비율
5. **업그레이드 우선순위** — 대개 여기서 티어가 갈린다 (아래 3장)

## 1. 종족별 시그니처 마이크로 (정확한 메커닉)

### 저그 — 뮤탈리스크

- **스택(매직박스)**: 뮤탈을 오버로드나 라바와 같은 컨트롤그룹으로 묶으면 겹쳐진다.
  - **대각선 이동**이 스택 유지가 잘 되고, **수평 이동**이 무빙샷(쏘고 빠지기)에 유리.
- **홀드 마이크로(핵심)**: 뮤탈 9기 이상이면 한 볼리에 **81뎀**. 마린 한 기(40HP)에 다 쏘면 **41뎀이 낭비**되므로 **2~3기에 분산**해서 쏘는 게 효율적. 'H' 키가 왼손에 가까워 실전에서 가장 자연스러운 방법.
- **바운스 데미지**: 뮤탈 공격은 튈 때마다 **÷3** (9 → 3 → 1). 그래서 뭉친 소형 유닛(마린·프로브)에 특히 강함. 어택 1업 시 9→10, 3→3.33... 즉 실효 증가분은 미미(≈1.44). **뮤탈전에선 어택보다 카라파이스가 훨씬 이득**.
- **스커지 회피**: 스타에서 가장 어려운 컨트롤 중 하나.
  - **차이니즈 트라이앵글**: 동쪽으로 도망 중이라면 **E→N→SE→W** 순으로 빠르게 클릭 → 스커지가 헷갈려 회전, 그 틈에 하나 스나이프하고 계속 도주.
  - **패트롤**을 뮤탈 무리 바로 앞(가까이)에 찍으면 감속 없이 반격.
  - 뮤탈을 **여러 경로로 나눠** 보내면 스커지가 어떤 뮤탈을 노리는지 보이므로, 안 노려진 뮤탈만 데려와 싸운다.
  - **절대 시야 안에 뮤탈을 idle로 두지 마라** — 상대가 스커지 클로닝을 쉽게 함. 자신 없으면 그냥 **뮤탈을 모아서** 붙어라(수가 많으면 스커지 클로닝 자체가 어려워지고 바운스에 죽음).
- **업그레이드 우선순위**: 뮤탈전 **카라파이스 3업 먼저(어택보다)**. 상대가 골리앗이면 **+1 아머 우선**(+1 어택보다).
- **역할별**:
  - **ZvT**: 상대 푸시를 최대한 지연 + 잡병/건물 스나이프 + 방어 빈틈 찌르기. 앞마당 성큰 1개 런바이 견제.
  - **ZvP**: **하이템플러 스나이프**가 1순위(스톰 무력화). 옵저버·프로브·캐논도 빠르게 제거.
  - **ZvZ**: 뮤탈+스커지 **제공권 싸움이 곧 승패**.

### 테란 — 벌처 + 스파이더 마인

- **패트롤 무빙샷(벌처 마이크로)**: 패트롤은 쿨다운이 없어 이동+발사+후퇴를 감속 없이 가능.
  1. 공격 대상 **뒤쪽으로 우클릭**(전진 방향 지정)
  2. **벌처 사거리 안, 특정 각도에 패트롤** → 벌처가 약 **15°만 틀며** 발사(거의 안 도는 것처럼 보임). 벌처 **뒤에** 패트롤 찍으면 뒤돌아버리니 금지. 너무 가까이 찍으면 발사 안 함.
  3. 발사 즉시 **우클릭으로 후퇴**(감속 금지 — 감속하면 적이 따라붙어 한 대 맞음).
  - 이온부스터(속업) 있으면 더 크게 돌아 난이도↑. 속업 전이 오히려 쉬움.
- **마인 활용**:
  - **고연사 유닛엔 촘촘한 라인**(안 그러면 마인이 터지기 전에 죽음). 탱크 라인과 조합하면 최소한 지연은 됨.
  - **정찰/지연엔 맵에 분산**(버로우 저글링처럼 상대 동선 파악).
  - **드라군 무리 포위**: 벌처로 드라군이 못 빠지게 막고 **사방에 마인** → 벌처는 마인 터지기 직전 후퇴(스플래시 자멸 방지).
- **마인 특성**: 깐 뒤 **약 18프레임 후 은폐**. 마인 위엔 건물을 못 짓고 착륙도 불가 → **확장 지연**용. 성큰은 벌처를 **3방**에 잡음(2방 맞고 런바이 가능하니 저그는 건물/저글링으로 길막).
- **참고**: 프로게이머 Flash의 벌처 마인 마이크로가 교본 수준.

### 프로토스 — 드라군 댄싱 / 리버 / 스톰

- **드라군 댄싱**: **싱귤래리티 차지(레인지업)** 필수(사거리 4→6). 레인지 드라군을 **마린 사거리 밖으로 살짝 빼고 홀드** → 아웃레인지 마린에게 공짜 샷 → 이 동작 반복.
  - 레인지 드라군은 **언업 마린이 지키는 벙커도 안 맞고** 때릴 수 있음(끝사거리로 조금씩 전진). 스팀 마린은 드라군 **3방**에 죽음.
  - 드라군은 폭발형 공격이라 **소형(저글링·속업 질럿)에 약함** → 그래서 테란이 대프로토스에서 메카닉을 감.
- **사이오닉 스톰**: **3x3 범위, 112뎀을 8틱(0.33s당)** 에 걸쳐 준다. **중첩 안 됨**(같은 곳에 2개 = 무의미), 건물 무효, **자기 하이템도 피격**.
  - **ZvP에서 최강**(소형·저체력 다수가 녹음). **TvP에선 제한적**(탱크는 스톰 1방 생존). 럴커엔 스톰 후 질럿/드라군 1기로 마무리.
  - **스톰 드랍**: 셔틀로 하이템 1기를 미네랄 라인에 내려 일꾼 학살.
- **리버 + 셔틀**: 스캐럽 **사거리 8**(대부분 방어건물보다 김), 스캐럽 **15미네랄**, **5기(업 시 10기)** 저장. 리버는 매우 느리고 **아머 0**이라 반드시 셔틀 조합.
  - **셔틀 픽업 타이밍**으로 드라군 샷 회피가 핵심. **스톰이 나오면** 셔틀+리버가 같이 맞아 리버플레이 하락.
  - 테란전 드랍 시 **질럿을 먼저 떨궈** 탱크 샷/마인 유도 후 리버 작업. 리버 2기를 번갈아 쏘면 끊김 없는 포화.
  - 리버 마이크로는 프로게이머 **SnOw**가 최고로 평가됨.

### 범용 마이크로 개념 (모든 종족)

- **무빙샷(카이팅)**: 이동↔공격(홀드/패트롤)을 리듬 있게 번갈아 → 사거리·속도 우위로 손해 없이 딜. 후퇴/추격/런바이에 사용.
- **자군 스플래시로 클로킹 잡기**: 탱크로 럴커 위 메딕 쏘기, 아콘/리버로 럴커 위 질럿 쏘기 등(디텍터 없을 때).
- **플랭크(포위)**: 교전 전 병력을 넓게 펼쳐 스플래시(마인/탱크/스톰) 피해 최소화.

## 2. 매치업 핵심 — 승리 조건 & 결정 지점

각 매치업은 "**핵심 아이디어 / 표준 흐름 / 승리 조건 / 흔한 실수**" 로 정리.

### PvZ (프로토스 vs 저그)
- **핵심**: 프로토스는 **반응형**이다. 대부분의 빌드가 저그의 공격 오프닝을 받아치게 설계돼 있고, 저그의 선택마다 트랜지션으로 **약점을 찌른다**(반응형 ≠ 수동적).
- **표준**: 패스트 익스팬드(FE) → 커세어/다크템플러/리버·하이템. 단단한 앞마당 **월인**이 전제.
- **비수빌드**: FE 후 **커세어+다크템플러** 조합(2006 GOMTV MSL S1 결승, 비수가 사비올에 3-0). PvZ 판도를 바꾼 상징적 빌드.
  - **커세어는 3역할**: ① 정찰 ② 뮤탈 방어 ③ 맵 장악. **오버로드와 교환하지 말 것**(뮤탈 방어용으로 아껴야).
  - **카운터**: **3베이스 스파이어 → 5햇 히드라**. 오버로드가 안전해져 커세어를 무력화 → 원조 비수빌드는 현대에선 제한적.
- **5햇 히드라 버스트 방어**: 대개 **18~24 히드라 + 속업 오버로드** 또는 **12~16 속업/사거리 히드라**로 온다. **하이템을 캐논 근처에 두고 스톰 준비**, 하이템 잃지 말 것. **떠도는 오버로드를 스나이프**하면 버스트를 크게 지연.
- **흔한 실수**: 커세어를 잡병 잡다 잃음 / 뮤탈 스나이프에 하이템 헌납 / 월인 부실로 저글링 난입.

### PvT (프로토스 vs 테란)
- **핵심**: 드라군 사거리로 **아웃마이크로 + 포지셔닝** 싸움. 프로토스 유닛이 개별로는 강하지만, **풀업 테란이 수가 많으면 더 효율적**.
- **표준 오프닝**: **드라군 + 옵저버**(옵저버가 마인 제거). 이후 **드라군/질럿** → 하이템/리버/아비터.
- **핵심 규칙**: 푸시를 막으려면 **프로토스 게이트웨이 = 테란 팩토리 + 2** 이상이어야 큰 공격+충원을 감당.
- **다크템플러 하라스**: 테란이 디텍션(터렛/컴셋)에 자원을 쓰게 강요 → 초반 압박 지연.
- **승리 조건**: 마인/탱크 라인을 **플랭크로 뚫기**(아래 푸시 브레이킹) + 아비터로 리콜/스테이시스.
- **푸시 브레이킹(질럿 버스트)**: 넓게 플랭크 → 드라군 어택무브로 마인밭 진입 → **질럿 2~3기 먼저 보내 마인 드래그** → 질럿 무리를 탱크 개별에 붙임 → 드라군은 스플래시 피해 벌리며 탱크 포커스.
- **흔한 실수**: 게이트:팩토리 비율 무시하고 무리한 진출 / 옵저버 없이 마인밭 돌진 / 하이템을 벌처에 스나이프당함.

### TvZ (테란 vs 저그)
- **핵심**: **메카닉(탱크/벌처/골리앗)** 이 주류. 벌처로 하라스·맵 컨트롤, 탱크 라인으로 전진, 골리앗/터렛으로 뮤탈 방어.
- **표준**: 1팩 더블 또는 2팩 벌처 → 마인/속업 → 탱크 추가 → 멀티. 벌처 1기로 앞마당 런바이 견제(성큰 3방).
- **뮤탈 대응**: 벌처는 초반 공중 무력 → **사이언스 베슬 이라디에이트**(뭉친 뮤탈에 치명적) + 골리앗/터렛. 뮤탈 상대 시 **+1 아머가 +1 어택보다** 나을 때 많음(저그 시점).
- **승리 조건**: 벌처 하라스로 저그 앞마당/드론을 갉으며 **탱크 라인을 안전하게 전진** → 멀티 견제. 후반 디파일러(다크스웜/플레이그) 나오면 벌처·베슬로 대응.
- **흔한 실수**: 뮤탈에 SCV·탱크 스나이프 허용 / 벌처를 저글링에 그냥 박음(마이크로 안 함) / 디텍션 없이 럴커밭 진입.

### ZvT / ZvP (저그 시점 요약)
- **ZvT**: **2햇/3햇 뮤탈** 오프닝 → 뮤탈로 푸시 지연·하라스 → **럴커/디파일러**로 후반. 성큰으로 벌처·초반 방어. **디파일러 다크스웜**이 대메카닉 핵심(탱크 딜 무효). 목표는 **테란이 멀티를 많이 못 먹게** 뮤탈 기동력으로 압박.
- **ZvP**: **하이템 스나이프**(뮤탈) + 히드라 다수 + 디파일러. 스톰이 최대 적 → **스톰 회피(빠른 반응+예측)** 가 필수 기술. 럴커로 앞마당 압박, 스커지로 커세어/셔틀 견제.

### TvT (테란 미러)
- **핵심**: 탱크/벌처/골리앗 시즈 라인 싸움 + 드랍(드랍십). 벌처는 초반 정찰·마인·컨테인. 후반 레이스/베슬.
- **승리 조건**: **시즈 라인 우위 + 드랍으로 멀티/일꾼 타격**. 포지션(하이그라운드 시즈)과 마인 라인이 핵심.

### PvP (프로토스 미러)
- **핵심**: **드라군 사거리 카이팅 싸움**(레인지·스피드 우위로 질럿 무한 카이팅). 리버/다크템플러/셔틀 견제.
- **리버 = 최우선 타겟**(고딜+스플래시로 판을 뒤집음). **리버 드랍 대비 드라군 3기 본진 상주**.
- 후반 스톰 나오면 리버·셔틀 마이크로 가치 급감(스톰에 같이 죽음).

### ZvZ (저그 미러)
- **핵심**: **뮤탈+스커지 제공권**이 곧 승패. 9풀 속업 저글링 압박 → 스파이어. 카라파이스 업글이 뮤탈전 우위.
- **승리 조건**: 스택·스커지 클로닝·뮤탈 마이크로 우위로 **상대 뮤탈을 먼저 잡고** 드론/성큰 밀기.

---

## 3. 업그레이드 · 조합 우선순위 (여기서 티어가 갈린다)

- **뮤탈전**: 카라파이스 3업 > 어택 (바운스 특성상 어택 실효 미미).
- **뮤탈 vs 골리앗**: +1 아머 > +1 어택.
- **바이오닉 테란**: +1 어택(마린 뭉치면 어택 스택 큼)과 사거리(U-238)·스팀 우선.
- **메카닉 테란**: 탱크 시즈텍 최우선 → 벌처 속업/마인 → 골리앗 사거리(대공).
- **프로토스**: **드라군 사거리(싱귤래리티)** 를 거의 항상 먼저 → 질럿 속업(대저그·후반) → 지상 공/방.
- **조합 원칙**: 모든 유닛엔 직접 카운터가 있으니 **혼합**으로 딜 최적화. 예) 질럿(탱커/마인받이) + 드라군(딜/대공) + 하이템(스플래시) + 리버(공성).

## 4. 레벨업 사다리 — 티어별로 뭘 고쳐야 하나

코칭 대상의 현재 티어를 파악하고 **그 티어의 병목**만 짚는다(상위 티어 조언은 아직 무의미).

- **입문(E~D)**: 서플 블록 안 하기 / 일꾼 계속 뽑기 / 표준 오프닝 하나를 손에 익히기 / 정찰 보내기.
  → 아직 마이크로 디테일 코칭은 이르다. **매크로 루틴**만.
- **중급(C~B)**: 정찰 정보로 **빌드 바꾸기**(반응) / 앞마당·확장 타이밍 / 기본 무빙샷·스택 / 업그레이드 챙기기.
  → 여기서부터 "정찰만 하고 안 바꿈"이 최대 병목.
- **고급(A~)**: 시그니처 마이크로(드라군 댄싱/벌처 패트롤/뮤탈 홀드) / 플랭크·포지셔닝 / 멀티태스킹(하라스+본진 생산 동시) / 타이밍 러시 정밀도.
- **프로급**: 심리전·빌드 메타 예측 / 프레임 단위 최적화 / 맵별 최적 빌드.

## 5. 리플레이 진단 체크리스트 (앙코르 코치 리포트용)

리플레이(screp) 데이터로 **규칙 기반** 자동 진단이 가능한 항목:

- **APM/EAPM** 추이 — 교전 구간 손 속도, 초반 빌드 구간 유휴.
- **서플 블록 구간** — 언제 몇 초 막혔나(빌드 지연 직결).
- **일꾼 생산 끊김** — 프로브/드론/SCV 생산 공백 타임라인.
- **첫 확장 타이밍** vs 매치업 표준(밀렸으면 왜 — 압박 대응? 아님 게으름?).
- **가스 타이밍 / 유휴 자원** — 미네랄·가스가 특정 임계 이상 쌓인 구간.
- **업그레이드 시작 타이밍** — 위 3장 우선순위 대비.
- **병력:일꾼 비율** — 특정 시점 과투자/과소투자.
- **교전 결과** — 병력 손실 교환비(무엇에 무엇을 잃었나).

이 항목들을 매치업 표준값과 비교해 **"6:00 서플 블록 8초 + 앞마당 표준보다 40초 지연 → 초반 압박이 없었는데도 늦음"** 처럼 **구체적 문장**으로 뽑으면, 진부한 코치 리포트가 아니라 실전 진단이 된다.

## 6. 프로급 빌드오더 & 타이밍 (실제 수치)

> "빌드오더 리스트"는 지시문일 뿐 — **왜 그 타이밍인지**를 이해해야 응용이 된다.
> 표기: 저그의 "12 해처리"는 **해당 건물을 지을 자원이 될 때의 서플(인구) 수**.

### 정찰 표준 타이밍 (코치 기준)
- **프로토스**: 첫 **파일런** 직후 프로브 정찰.
- **테란**: 첫 **서플 디팟** 직후 SCV 정찰.
- **저그**: 첫 **오버로드 2기** + 초반 드론 1기(10~12 서플)로 정찰.
- → 이보다 한참 늦으면(예: 4:00 이후 첫 정찰) 빌드 파악이 늦어 대응이 꼬인다.

### 저그 3해처리 오프닝(경제형)
```
12  해처리 (앞마당)
11  스포닝 풀
13  해처리 (3번째)
16  익스트랙터
```
경제는 강하지만 초반 올인/러시에 취약 → 상대 정찰로 성큰 타이밍 조절.

### 2해처리 뮤탈 vs 테란 (ZvT)
- **핵심**: 공격성이 생명. 뮤탈로 **맵 컨트롤 장악**, 테란의 히브 전 단 한 번의 타이밍 푸시만 조심. 유닛 잃고 맵 컨트롤 뺏기면 진 거다.
- **12 해처리 변형**: 경제 좋음, 뮤탈이 완벽 시 **약 6:30**. 저글링 압박 옵션 있음(초보에 더 적합).
- **12 풀 변형**: 뮤탈이 **약 6:00**로 더 빠름(허 찌르기). 대신 경제 극도로 빡빡 — 조금만 삐끗해도 스파이어 완성 시 뮤탈 6기 미만. 저글링 하라스·성큰이 뮤탈을 지연시킴.
- **전환**: 뮤탈 11기 뒤 **히드라 덴 → 에볼루션 챔버 순서**로 짓고 럴커/링/스컬지. **+1 카라파이스** 중요(피해량에 따라 타이밍 조절). 데미지 크게 줬으면 퀸즈네스트 먼저 → 가디언/럴커/디파일러 + 나이더스로 디파일러 이동.
- **테란의 공격 창(역이용)**: 테란은 FE 후 2개의 최적 공격 타이밍 — ① 저그가 **뮤탈→럴커 전환**할 때(마린메딕 푸시), ② 이후 베슬+탱크로 럴커 카운터(디파일러 다크스웜 전까지). 이 창을 알고 방어를 준비.

### 2해처리 뮤탈 vs 프로토스 (ZvP)
- **핵심**: 올인 아님. 경제 피해 못 줘도 최소한 프로토스를 수비적으로 묶으면 성공. 뮤탈로 **커세어·외딴 캐논·하이템·드라군·프로브** 스나이프.
- **요령**: 가스를 3번째 해처리보다 먼저, 뮤탈 6기 최속. **풀 300HP에서 유닛 생산 중단**(라바 3개 남겨 저글링). **스파이어 350HP 직전 오버로드 2기**(라바 6개로 뮤탈). 초반 저글링 8기 이후엔 경제상 더 못 뽑음 → 런바이 위협·정찰 차단·외딴 질럿 정리에만 사용.
- **주의**: 커세어/리버 상대는 힘든 싸움(커세어가 뮤탈 카운터). 더블 스타게이트면 크리티컬 매스 2배 빨리 참 → 뮤탈/스컬지 마이크로로 커세어 수를 억제하며 히드라 전환.
- **전환**: 뮤탈 11기 뒤 히드라 덴 → 럴커/링/스컬지 또는 히드라/링/뮤탈. **에어 카라파이스 먼저** 업글.

### (요약) 나머지 매치업 표준 뼈대
- **TvP**: 1팩 더블 or 2팩 → 벌처 속업/마인 → 탱크. **페이크 더블**은 마인으로 드라군 견제하며 앞마당. 드라군+옵저버 상대.
- **PvT**: 드라군+옵저버 오프닝(옵저버로 마인 제거) → 드라군/질럿 → 하이템/리버/아비터. **게이트 = 팩토리+2** 유지.
- **PvZ**: FFE(포지 퍼스트, 좁은 입구 캐논 방어) 또는 게이트 확장 → 커세어/다크템/하이템. **비수빌드** 참고(1장).
- **ZvZ**: 9풀 속업 → 스파이어. **뮤탈+스컬지 제공권**이 전부. 카라파이스 우위.
- **TvT / PvP**: 각각 탱크 시즈 라인 / 드라군 사거리 싸움. 드랍·리버 견제로 균형을 깬다.

## 7. 코치 리포트 도구 (앙코르 레코더)

레코더에 **규칙 기반 코치 리포트**가 들어갔다(진부한 조언이 아니라 **상대 대비 구체적 진단**).

```
sc_recorder.exe --coach <리플레이.rep>
```

리플레이(screp) 데이터로 자동 진단하고, **가장 심각한 항목부터** 뽑아준다:
- 생산 공백(최대 N초) / 생산 가동률(%)
- 일꾼 수 **상대 대비** 차이 (예: "프로브 24기 적음 34 vs 58")
- 앞마당·멀티 타이밍 **상대 대비** (예: "앞마당 220초 늦음 6:20 vs 2:40")
- 초반 정찰 여부/시각 (표준: 첫 파일런/서플/2오버로드 직후)
- 공/방 업그레이드 **상대 대비** (예: "공0방0 vs 공3방2")
- **매치업별 조합/테크** 점검 (PvZ 하이템·아콘, TvZ 베슬, ZvT 디파일러 등)

예시 출력:
```
Me (PvZ) vs Zerg  |  패배  |  길이 18:40
  1. 생산이 최대 52초 끊겼어요 — 자원이 남으면 즉시 큐를 채우세요.
  2. 프로브가 상대보다 24기 적어요(34 vs 58). 초반 일꾼 끊김을 점검하세요.
  3. 상대 본진을 초반에 정찰하지 못했어요.
  4. 공/방 업그레이드가 상대보다 낮아요(공0방0 vs 상대 공3방2).
  ...
```

향후 이 리포트를 앙코르 웹 갤러리의 경기 상세에 붙이면, 각 판마다 "무엇을 고쳐야 하는지"가 자동으로 뜬다.
