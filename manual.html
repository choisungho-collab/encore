Get Started

스타 켜고 게임만 하세요.
나머지는 자동입니다.

프로그램을 켜두면 브루드워 게임이 끝날 때마다 실제 게임 화면(HD)이 자동으로 녹화되고, 리플레이가 분석돼 갤러리에 쌓입니다.

패키지 받기 (.zip)Windows · 압축 풀고 더블클릭이면 끝

1다운로드

위 버튼으로 전체 패키지(zip)를 받습니다. 압축을 풀면 폴더가 하나 나와요. 클라우드 설정은 이미 들어있어서 따로 만질 게 없습니다.

2실행

압축 푼 폴더 안에서 아래 둘 중 하나를 더블클릭하세요.

START_no_console.vbs추천

검은 콘솔 대신 상태 창(앱)이 떠서 녹화 상태·경기 수·로그를 보여줍니다. 브라우저에 갤러리도 같이 열려요.

START.bat

콘솔 창과 함께 실행됩니다. 문제가 생겼을 때 원인(로그)을 보기 좋아요.

처음 실행은 ffmpeg를 자동으로 받느라(약 90MB, 1~2분) 갤러리가 조금 늦게 열릴 수 있어요. 한 번만 받으면 다음부턴 바로 켜집니다.

3필요한 것

  • Python 3 — 없으면 python.org 에서 설치 (설치 시 "Add to PATH" 체크)
  • StarCraft: Remastered — 브루드워 (SC2 아님)
  • NVIDIA GPU면 자동으로 하드웨어 인코딩(NVENC)을 써서 게임 부담이 적어요. 안 되면 자동으로 소프트웨어로 전환됩니다.

설정은 폴더 안 config.json 에서 바꿀 수 있어요 (자세한 건 매뉴얼 참고).
